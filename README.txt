Instructions:
Login from the "Login" menu using credentials. Navigate the Appointment, Customer and Reports menu with the buttons on top; each of these menu's has their own features for use through buttons. Log Off to return to the login menu.

Additional Report:
My additional report lists the customers filtered by country in a tableview.
